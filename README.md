# Library_fastAPI

Реализовано:
  - контейнерезация: сервер под fastAPI с фронтэндом в своем контейнере, база mysql в своем.
  - фронтэнд на JS
  - middleware на python (fastAPI)
  - база данных mysql
  - взаимодействие между контейнерами между фронтэндом (JS), мидллвейр (python) и mysql
